
# ТЗ: SEO-пересборка трёх коммерческих страниц сайта РУМЯНА БЭНД

## ЦЕЛЬ
Создать три полноценные посадочные страницы вместо thin-контента:

- kaver-gruppa-na-svadbu.html
- kaver-gruppa-na-korporativ.html
- kaver-gruppa-na-yubiley.html

Страницы должны:
- Устранить BAD_QUALITY / thin content
- Иметь высокий CTR в поиске
- Быть уникальными по интенту
- Использовать существующий качественный контент с cover.html, folk.html, rusalki.html и главной

## ИСТОЧНИКИ КОНТЕНТА (обязательно использовать)
- https://rumyana.su/cover (основной кавер 4–8 человек — главный источник продающих текстов, сценария, преимуществ, видео)
- https://rumyana.su/folk (фолк-элементы: баян, балалайка, народные обработки)
- https://rumyana.su/rusalki (камерный/нежный формат, авторские песни, многоголосие)
- Главная https://rumyana.su/ (hero-стиль, общие преимущества, видео)

## ГЛАВНОЕ ПРАВИЛО
Каждая страница — отдельный пользовательский интент.  
НЕ копировать тексты один в один между страницами. Адаптировать акценты, примеры, отзывы и формулировки.

## ОБЯЗАТЕЛЬНАЯ СТРУКТУРА КАЖДОЙ СТРАНИЦЫ (в этом порядке)

1. **<head>**  
   - Уникальный `<title>` (60–70 символов, ключ в начале)  
   - Уникальный `<meta name="description">` (150–200 символов, продающий)  
   - Опционально `<meta name="keywords">`

2. **Hero-блок**  
   - Большой уникальный H1  
   - Подзаголовок (1–2 предложения)  
   - 3–4 CTA: Написать в Telegram, Позвонить, Смотреть промо/видео

3. **Видео-блок** (минимум 3–4 видео в сетке)  
   - Использовать реальные видео с cover/folk/rusalki (свадьба/корпоратив/юбилей где возможно)  
   - Каждое видео с коротким заголовком

4. **Описание под интент** (2–4 абзаца, продающий текст)

5. **Почему нас выбирают** (5–7 преимуществ с иконками/эмодзи) — адаптировать под интент

6. **Как проходит выступление / Сценарий** (5 этапов: Welcome → Первый блок → Интерактив → Смена образов → Финал) — слегка адаптировать текст

7. **Галерея** (8–15 фото с правильными `alt`-атрибутами, содержащими ключи)

8. **Состав и стоимость**  
   - Минимальный / Оптимальный (выделить) / Полный состав  
   - Цены из cover (от 45 000 ₽ и выше) + пояснение под тип мероприятия

9. **Дополнительные опции**

10. **Отзывы и кейсы** (минимум 3–4 уникальных отзыва, разные для каждой страницы)

11. **Частые вопросы (FAQ)** (6–8 вопросов + ответов, релевантных именно этой странице)

12. **SEO-текст внизу** (1800–2800 символов) — читаемый, с естественным вхождением ключей, без переспама

13. **Перелинковка**  
    - Ссылки на две другие страницы услуг  
    - На главную, /cover, /folk, /rusalki

Задача:

* повысить релевантность под поисковые запросы

## ❗ ГЛАВНОЕ ПРАВИЛО

Каждая страница = отдельный интент.

НЕЛЬЗЯ:

* копировать одну страницу в три
* делать одинаковые тексты
* использовать одинаковые H1/Title
* делать шаблон без адаптации

## 📄 СТРУКТУРА СТРАНИЦЫ (ОБЯЗАТЕЛЬНА)

Каждая страница должна содержать ВСЕ блоки ниже:

---

### 1. HERO-БЛОК

Содержит:

* H1 (уникальный)
* подзаголовок
* кнопки CTA

Пример структуры:

```
<h1>...</h1>
<p>...</p>
<a>Написать в Telegram</a>
<a>Позвонить</a>
<a>Смотреть видео</a>
```

---

### 2. ВИДЕО-БЛОК

Требования:

* минимум 2 видео, максимум 4
* сетка (grid), не один плеер
* реальные видео выступлений

---

### 3. ОПИСАНИЕ ПОД ИНТЕНТ

Текст 2–4 абзаца.

Разный смысл для каждой страницы:

СВАДЬБА:

* эмоции
* первый танец
* атмосфера

КОРПОРАТИВ:

* вовлечение гостей
* работа с ведущим
* динамика

ЮБИЛЕЙ:

* персонализация
* любимые песни
* душевность

---

### 4. БЛОК ПРЕИМУЩЕСТВ

Минимум 5 пунктов:

* вокал
* образы
* интерактив
* репертуар
* режиссура

Текст адаптировать под страницу.

---

### 5. СЦЕНАРИЙ ВЫСТУПЛЕНИЯ

5 этапов:

1. Welcome
2. Первый блок
3. Интерактив
4. Смена образов
5. Финал

Можно использовать один шаблон, но текст слегка менять.

---

### 6. ГАЛЕРЕЯ

Требования:

* 8–15 изображений
* обязательно атрибут alt

Примеры alt:

* "кавер группа на свадьбу москва"
* "живая музыка на корпоратив"
* "группа на юбилей выступление"

---

### 7. СТОИМОСТЬ

Показать:

* минимальный состав
* оптимальный состав (выделить)
* диапазон цен

Текст адаптировать:

СВАДЬБА → сценарий дня
КОРПОРАТИВ → бюджет компании
ЮБИЛЕЙ → гибкость формата

---

### 8. ДОПОЛНИТЕЛЬНЫЕ ОПЦИИ

Список:

* саксофон
* акустика
* флешмобы
* индивидуальная программа

---

### 9. ОТЗЫВЫ / КЕЙСЫ

Минимум 3 отзыва.

ВАЖНО:

* тексты разные для каждой страницы
* не копировать

---

### 10. FAQ

Минимум 5 вопросов.

Примеры:

СВАДЬБА:

* можно ли первый танец?
* работаете ли с ведущим?

КОРПОРАТИВ:

* можно ли брендировать?
* работаете ли с агентствами?

ЮБИЛЕЙ:

* можно ли выбрать песни?
* подходит ли для 50+ аудитории?

---

### 11. SEO-ТЕКСТ (НИЗ СТРАНИЦЫ)

1500–2500 символов.

Обязательно включить ключевые фразы:

* кавер группа на свадьбу москва
* кавер группа на корпоратив москва
* группа на юбилей
* живая музыка на мероприятие

Текст:

* без воды
* без переспама
* читаемый
